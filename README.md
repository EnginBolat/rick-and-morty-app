## Run Locally

Clone the project

```bash
  git clone https://github.com/EnginBolat/rick-and-morty-app.git
```

Go to the project directory

```bash
  cd rick-and-morty-app
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
